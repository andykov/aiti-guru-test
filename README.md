# Aiti Guru - Тестовое задание

Административная панель управления товарами.

## Стек

- React 18 + TypeScript (strict mode)
- Vite
- Tailwind CSS v4
- shadcn/ui (Radix UI)
- Zustand (state management)
- TanStack Query (data fetching)
- React Router v7
- React Hook Form + Zod (forms + validation)
- Axios (HTTP client)

## Запуск

```bash
npm install
npm run dev
```
